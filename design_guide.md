# ✨ Style Guide: CleanOps

## Aesthetic: Premium Dark mode
- **Primary Background**: `#0b0f1a` (Deep Navy)
- **Secondary Background**: `#111827` (Matte Black/Glass)
- **Accents**: `#6c63ff` (Electric Purple/Blue Gradient)
- **Glassmorphism**: 
  - `backdrop-filter: blur(20px)`
  - `background: rgba(17, 24, 39, 0.7)`
  - `border: 1px solid rgba(255, 255, 255, 0.07)`

## UI Components
- **Buttons**: Rounded (12px), gradients, subtle outer glow.
- **Inputs**: Custom icons, high-contrast focus rings.
- **Feedback**: Confetti on completion, smooth scale transitions for cards.

## UX Principles
- **Speed**: One-handed mobile use (< 10 seconds to core action).
- **Clarity**: Status indicators (pill-style) using semantic colors (Green/Blue/Yellow).
